## Why TabeebPlus

TabeebPlus is not just clinic management software — it is a **complete Clinic Operating System** that digitizes every workflow in a medical practice. Built by [FekraLabs](https://fekralabs.com) in Cairo, Egypt, it was purpose-designed for the Arabic-speaking healthcare market with full bilingual support (RTL Arabic + LTR English).

| Challenge | TabeebPlus Solution |
|-----------|---------------------|
| No centralized appointment system | Smart scheduling with conflict prevention, queue management, and patient self-booking |
| Paper-based medical records | Encrypted (AES-256) EMR with full patient timeline, vitals, lab results, and prescriptions |
| Complex billing and insurance | Automated invoicing, insurance claims, prior authorizations, and ZATCA e-invoicing (Saudi) |
| Doctor commission disputes | Transparent commission engine with customizable rules, retroactive recalculation, and settlements |
| Multi-branch chaos | Centralized dashboard with branch-scoped permissions and cross-branch patient records |
| Limited analytics | 19 specialized reports: financial, clinical, operational, and efficiency analytics |
| Weak patient engagement | WhatsApp reminders, patient portal, online booking, and in-app chat |
| No offline capability | Desktop application with local database backup and recovery |

> **TabeebPlus is the first platform to coin and own the "Clinic Automation" category in the Arabic-speaking world.**
> — [tabeebplus.com/en/clinic-automation](https://tabeebplus.com/en/clinic-automation)

## Key Features

### Clinical Operations

| Feature | Description |
|---------|-------------|
| **Electronic Medical Records (EMR/EHR)** | Complete patient timeline with vitals, allergies, chronic diseases, lab results, and visit history — all encrypted |
| **Appointment Scheduling** | Smart calendar with conflict prevention, recurring appointments, queue management, and real-time status tracking |
| **E-Prescriptions** | Digital prescriptions with customizable print templates, drug interaction alerts, and allergy warnings |
| **Clinical Examination** | Dedicated examination interface with vital signs capture, diagnosis coding, lab/radiology orders, and prescription presets |
| **Queue Management** | Branch-level queue counters with auto-incrementing turn numbers and late-patient tracking |
| **Patient Portal** | Self-service portal for patients to book appointments, view records, and manage their profile |

### Financial Management

| Feature | Description |
|---------|-------------|
| **Medical Billing and Invoicing** | Automated invoice generation, payment tracking, installment plans, and patient ledger |
| **Insurance Claims** | Full insurance lifecycle: company management, tariff rates, prior authorization, claim batches, and settlements |
| **Doctor Commission Engine** | Flexible rules (percentage, fixed, tiered, service-based) with retroactive recalculation and bulk settlements |
| **Expense Tracking** | Categorized expense management with approval workflows and financial reports |
| **ZATCA E-Invoicing** | Phase 2 compliant electronic invoicing for Saudi Arabia clinics |

### Operations and Administration

| Feature | Description |
|---------|-------------|
| **Multi-Branch Management** | Centralized control of multiple clinic locations with branch-scoped permissions |
| **Employee and Role Management** | Granular permission system with hierarchical role-based access control (RBAC) |
| **Doctor Schedules** | Weekly schedule management with multiple daily shifts, max patient limits, and absence handling |
| **Inventory Management** | Medical supply tracking with batch numbers, expiry alerts, supplier management, and dispensing |
| **Audit Logs** | Complete audit trail of all system actions for compliance and security |
| **Data Export** | Export to XLSX, CSV, PDF, JSON with scheduled recurring exports and email delivery |

### Communication and Engagement

| Feature | Description |
|---------|-------------|
| **Internal Chat** | Real-time messaging with file attachments, reactions, read receipts, and conversation linking to patient records |
| **Push Notifications** | Browser push notifications with real-time streaming |
| **WhatsApp Integration** | Appointment reminders and patient communication via WhatsApp |
| **Blog and CMS** | Built-in bilingual content management system for medical blogs and landing pages |

### Specialized Clinical Modules

| Feature | Description |
|---------|-------------|
| **205+ Medical Calculators** | Peer-reviewed clinical decision tools across 20 specialties with PubMed references |
| **296 Disease Hubs** | ICD-10 coded disease reference pages with related calculators and clinical information |
| **Telemedicine** | Encrypted video consultations linked directly to patient EMR records |
| **Dental Charting** | Interactive dental charting with tooth-level annotations |
| **OB/GYN Module** | Antenatal care tracking, gestational age calculation, and fetal weight estimation |
| **Pediatrics** | Growth tracking, vaccination schedules, and pediatric-specific calculators |

## Architecture

TabeebPlus is built on a modern full-stack framework with a multi-tenant SaaS architecture. The platform serves thousands of concurrent users across multiple clinic branches with real-time data synchronization, role-based access control, and enterprise-grade security.

**Key architectural pillars:**

| Pillar | Description |
|--------|-------------|
| **Multi-Tenant** | Each clinic operates in an isolated tenant with scoped data, permissions, and branding |
| **Role-Based Access** | Granular permission system with organization, branch, and global scopes |
| **Real-Time** | Server-Sent Events for live notifications, chat, and queue updates |
| **Offline-Capable** | Desktop application with local database and automatic sync |
| **Bilingual** | Full Arabic (RTL) and English (LTR) support at every layer |
| **AI-Ready** | Native AI agent discovery, MCP tools, and structured data for LLM citations |
